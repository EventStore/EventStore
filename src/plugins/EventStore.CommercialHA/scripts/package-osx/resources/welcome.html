Welcome to EventStore!

Event Store is a domain specific database for Event Sourcing

Installation note

Please take note of the following important files and directories:
  • /usr/local/bin/eventstored - The EventStore daemon
  • /etc/eventstore/eventstore.conf - The EventStore configuration file
  • /var/lib/eventstore - The path where data files will be stored by default
  • /var/log/eventstore - The path where log files will be stored by default

    • Starting and stopping the EventStore service

    Once installed, the EventStore service will launch automatically. The Web UI will be accessible on http://localhost:2113

    Starting the EventStore service 
    sudo launchctl start org.eventstore.eventstore-db

    Stopping the EventStore service 
    sudo launchctl stop org.eventstore.eventstore-db
